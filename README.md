# Aurora Bot

Aurora is a bot for collect bonus boxes in Darkorbit

This bot is under development

## Features

- [x] Collect BonusBoxs
- [x] Revive on Portal
- [x] Revive on Base
- [x] Back to map
- [x] Escape when is under attack
- [x] Use Clo4cks
- [x] Repair PET (Destroyed)
- [ ] Repair PET (HP)
